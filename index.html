Отправить на сайт

Что нового на этом устройстве по сравнению с сайтом:

1. github.com → аватарка → Settings → внизу Developer settings.
2. Personal access tokens → Fine-grained tokens → Generate new token.
3. Имя: souffleur. Expiration: Custom → на год.
4. Repository access: Only select repositories → выбери souffleur.
5. Permissions → Repository permissions → Contents → Read and write.
6. Generate token → скопируй ключ github_pat_… (показывается один раз) → вставь сюда.
Ключ хранится только на этом устройстве, как пароль.

Новая песня

Ткни в текст → «▣ Блок» выделит абзац (или собирай выделение кнопками ±строка) → жми метку. Отступы сделаю сам, куплеты нумеруются сами, старая метка заменяется новой.
Пустая строка = отступ между блоками. Метки вроде [Припев], [Куплет 2], [Бридж] будут выделены цветом на сцене.
